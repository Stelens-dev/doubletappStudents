## Разработанные функции приложения:
- Отображение списка студентов с полями: Фото, ФИО, специальность, группа, возраст, рейтинг, любимый цвет;
- Удаление студента из списка;
- Поиск студента по ФИО;
- Сортировка по ФИО, рейтингу, возрасту и любимому цвету.

## Как запустить в dev режиме?
1. Установить Node.js (Версия которая использовалась в проекте v20.13.1)
2. Установить проект
```sh
git clone https://github.com/Stelens-dev/doubletappStudents.git
```
3. Установить зависимости
```sh
npm i
```
4. Запустить проект
```sh
npm run dev
```

## Как запустить в build режиме?
1. Сделать все те же шаги, которые были в этом разделе *[Как разворачивать в dev режиме?](#как-запустить-в-dev-режиме)* с 1 по 3 пункт;
2. Запустить проект
```sh
npm run build
```
